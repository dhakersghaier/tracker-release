# TimeTracker releases

Public builds for TimeTracker. Source code is private.

## macOS install

```bash
curl -fsSL "https://raw.githubusercontent.com/dhakersghaier/tracker-release/main/install-macos.sh" | bash
```

Or double-click [TimeTracker-Install.command](./TimeTracker-Install.command).

## Downloads

- [macOS DMG](./macos/timetracker-macos.dmg)
- [Windows installer](./windows/timetracker-setup.exe) (wizard + desktop shortcut)
- [Windows ZIP](./windows/timetracker-windows.zip) (fallback)

## Windows install

```powershell
irm https://raw.githubusercontent.com/dhakersghaier/tracker-release/main/install-windows.ps1 | iex
```

Or double-click [TimeTracker-Install.bat](./TimeTracker-Install.bat).
